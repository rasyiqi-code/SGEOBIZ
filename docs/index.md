## Apa itu SGEOBIZ SEO?

**SGEOBIZ SEO** adalah plugin WordPress yang mengotomasi seluruh kebutuhan SEO website Anda — mulai dari meta tag, sitemap, Open Graph, structured data Schema.org, hingga fitur khusus untuk bisnis lokal Indonesia seperti Google Business Profile (GBP) dan integrasi marketplace.

Plugin ini dibuat dengan prinsip: **lakukan lebih sedikit, raih hasil lebih baik.**

## Instalasi

### Persyaratan

- **PHP** 7.4 atau lebih baru
- **WordPress** 6.7 atau lebih baru
- Browser modern untuk administrasi

### Langkah Instalasi

1. Upload plugin ke direktori `/wp-content/plugins/`
2. Aktifkan plugin melalui menu **Plugin** di WordPress
3. Plugin langsung aktif — SEO dasar sudah dikonfigurasi otomatis
4. Kunjungi **SGEOBIZ SEO → Pengaturan** untuk konfigurasi lanjutan

---

## Fitur Unggulan untuk Indonesia

### 🏢 Profil Bisnis (Google Business Profile)
Isi data NAP (Name, Address, Phone), koordinat GPS, jam operasional, dan link semua platform digital bisnis Anda — Facebook, Instagram, TikTok, Tokopedia, Shopee, GoFood, dan lainnya.

### 📍 Schema Bisnis Lokal
Output structured data `LocalBusiness` secara otomatis ke halaman Anda untuk meningkatkan peluang muncul di Google Maps dan hasil pencarian lokal.

### 🛒 Integrasi Marketplace Indonesia
Dukung link ke Tokopedia, Shopee, Lazada, Blibli, Zalora dalam satu panel pengaturan terpusat.
